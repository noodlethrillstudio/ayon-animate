# ayon-animate
Animate integration for Ayon


Components: 

Server. This contains the majority of the settings. Anything that can be changed as system prefs for the studio go in here. 

Client: 
    Api: The Ayon side code. Contains the virtual webserver. Ayon integration acts as server and animate acts as client.
    
    Extension: The animate side code. Built in zxp format.
